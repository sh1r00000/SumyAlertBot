# 🚨 Sumy Alert Bot

Telegram-бот для автоматического оповещения о воздушных тревогах в Сумах и Сумской области.

## 📌 Возможности

- 🚨 Автоматическое уведомление о начале воздушной тревоги.
- ✅ Автоматическое уведомление об отбое тревоги.
- 📍 Отображение района или города, где объявлена тревога.
- ⏱ Отображение времени начала и продолжительности тревоги.
- 💾 Автоматическое сохранение активной тревоги в базе данных.
- 🔄 Восстановление состояния после перезапуска бота.
- ⚡ Работа в режиме 24/7.

## ⚙️ Установка

```bash
git clone <repository>
cd SumyAlertBot
python -m venv venv
```

Windows

```bash
venv\Scripts\activate
```

Linux

```bash
source venv/bin/activate
```

Установить зависимости

```bash
pip install -r requirements.txt
```

---

## 🔑 Создать файл `.env`

```
BOT_TOKEN=YOUR_BOT_TOKEN
CHANNEL_USERNAME=@your_channel
ALERTS_API_KEY=YOUR_API_KEY
```

python bot.py
