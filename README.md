# 🚨 Тривожні Суми

Telegram-бот для автоматичного сповіщення про повітряні тривоги та відбої у Сумській області.

## 📌 Можливості

- 🚨 Автоматичне сповіщення про початок повітряної тривоги.
- ✅ Автоматичне сповіщення про відбій тривоги.
- 📍 Відображення району або громади, де оголошена тривога.
- ⏱️ Відображення часу початку та тривалості тривоги.
- 💾 Автоматичне збереження активної тривоги у базі даних.
- 🔄 Відновлення стану після перезапуску бота.
- 🤖 Повністю автоматична робота без участі адміністратора.
- ☁️ Робота на Railway 24/7 без необхідності тримати комп'ютер увімкненим.
- ⚡ Миттєва публікація повідомлень у Telegram-канал.
