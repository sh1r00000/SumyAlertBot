# 🚨 Тривожні Суми

Telegram-бот для автоматичного сповіщення про повітряні тривоги та відбої у місті Суми та Сумському районі.

## 📌 Можливості

- 🚨 Автоматичне повідомлення про початок повітряної тривоги.
- ✅ Автоматичне повідомлення про відбій тривоги.
- 🏙️ Підтримка міста Суми.
- 🏛️ Підтримка Сумського району.
- ⏱️ Відображення часу початку та тривалості тривоги.
- 💾 Збереження активної тривоги у базі даних.
- 🔄 Відновлення стану після перезапуску Railway.
- ⚡ Робота у режимі 24/7.
- 📡 Використання офіційного API alerts.in.ua.

## ⚙️ Встановлення

```bash
git clone <repository>
cd SumyAlertBot

python -m venv venv
```

### Windows

```bash
venv\Scripts\activate
```

### Linux

```bash
source venv/bin/activate
```

Встановити залежності:

```bash
pip install -r requirements.txt
```
